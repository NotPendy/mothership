Mothership

The goal of this project is to develop and demonstrate a mothership setup with a fixed wing carrier (or airship ;-), capable of launching and recovering one or more smaller drones (probably multicopters).

To successfully complete this project you need to have:

  • At least two pilots (one for mothership and one for the little one).
  • Inclination to build complex systems.

Deliverables

  • Demonstration
  • Well commented code and schematics
  • Documentation for the resulting system
  • Report
