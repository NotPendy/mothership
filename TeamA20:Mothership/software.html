Environment on Drone

Raspberry Pi Lecacy OS "Debian Buster" Download
Python 3.9

Vision Dependencies

  • OpenCV-Python 4.5.5.64 `pip3 install opencv-python`
  • imutils 0.5.4 `pip3 install imutils`
  • Libatlas 3.10.12 or Most Recent Version `sudo apt-get install libatlas-base-dev`
  • Numpy 1.22 `pip3 install -U numpy`

Servo Dependencies

  • RPi.GPIO 0.7.1 `pip3 install RPi.GPIO`

Drone Dependencies

  • Dronekit-Python 1.55 `pip3 install dronekit`
  • pymavlink 2.4.29 `pip3 install pymavlink`
